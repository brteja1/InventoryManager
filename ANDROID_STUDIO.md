# Android Studio Import

This repo is already structured as a Gradle Android project.

## Open In Android Studio

1. Start Android Studio.
2. Choose `Open`.
3. Select the repository root folder: `InventoryManager`.
4. Let Android Studio sync the Gradle project.

## If Sync Prompts For Setup

- Install the Android SDK version requested by the project.
- Use the built-in Gradle wrapper from `gradlew`; do not switch to the system Gradle installation.
- If Android Studio asks for a local SDK path, point it at your Android SDK install.

## Expected Project Entry Points

- Gradle settings: `settings.gradle.kts`
- App module: `app/`
- Gradle wrapper: `gradlew`
